# ☕ Café Python - Proyecto Web Fullstack

Aplicación web profesional para gestión de pedidos en cafetería. Desarrollada con arquitectura MVC manual, Python nativo y JavaScript Modular.

## 📋 Características Principales
- **Frontend:** HTML5, CSS3, JavaScript (ES6 Modules).
- **Backend:** Python 3 (`http.server`), sin frameworks externos.
- **Base de Datos:** MySQL (con creación automática de tablas).
- **Funciones:** Login/Registro, Carrito de compras, Panel de Administración.

## 🚀 Instrucciones para Ejecutar

Sigue estos pasos para probar el proyecto en tu máquina local:

### 1. Requisitos Previos
- Tener instalado **Python 3.10** o superior.
- Tener instalado **XAMPP** (o cualquier servidor MySQL).

### 2. Configuración
1. Abre **XAMPP Control Panel**.
2. Inicia los módulos **Apache** y **MySQL** (Botones "Start").
3. Instala la librería conectora de Python abriendo tu terminal en la carpeta del proyecto:
   ```bash
   pip install mysql-connector-python
